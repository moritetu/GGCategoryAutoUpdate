This plugin helps you to update category automatically when you post an article from WordPress admin panel or xmlrpc post.

== Description ==

GGCategoryAutoUpdate can regist new category, set category to the post when you post an article from WordPress admin panel
or via xmlrpc request. You only have to write a syntax in the post content. For example:
`<!-- ggcat[category] -->`


== Installation ==

1. Upload the entire `gg-category-auto-update` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'GGCategoryAutoUpdate' in settings menu in your WordPress admin panel.

For basic usage, you can also have a look at the [plugin homepage](http://wiki.grepgrape.net/index.php?WordPress%2FGGCategoryAutoUpdate).

== Frequently Asked Questions ==

= Is category added newly if the specified category is yet to regist? =

Default is No, but you can allow it.
